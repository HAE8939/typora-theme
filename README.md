# Matoujie Theme

Matoujie is a clean, elegant Typora theme designed by HAE. It focuses on a calm, high-readability editing experience with warm, restrained colors.

## Features

- Clean, high-contrast design optimized for long-form writing
- Warm color palette with careful attention to readability
- Proper CJK font support with fallbacks
- Monospace font optimized for code blocks (Maple Mono NF CN)
- Balanced typography with comfortable line heights and spacing

## Installation

1. Open Typora and go to `Settings` → `Appearance` → `Open Theme Folder`
2. Copy `matoujie.css` into the theme directory
3. Restart Typora
4. Select `Matoujie` from the `Theme` menu

## Customization

Edit `matoujie.css` to customize:

- **UI Font**: Modify `--font-ui` at the top of the file
- **Mono Font**: Adjust `--font-mono` for code blocks
- **Colors**: Core colors are defined as CSS variables in `:root`
- **Layout**: Width and spacing can be fine-tuned in `#write`
