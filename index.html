Features
🔤

Font Isolation

Unicode-range separates Chinese and English rendering with independent @font-face.

⚖️

Weight Decoupling

Inter Regular (400) for English, Noto Sans SC Light (300) for Chinese — visually balanced.

🎨

Warm Palette

Clay + Ivory color system with careful attention to readability and mood.

📐

CSS Variables

Full token-based design system for easy customization.

🔗

Link Icons

External links auto-display icons, anchors excluded.

🖨️

Print Ready

Link icons suppressed in print, page breaks optimized.

Installation

1. Open Typora → Settings → Appearance → Open Theme Folder

2. Copy matoujie.css and the fonts/ directory

3. Restart Typora

4. Select Matoujie from the Theme menu
