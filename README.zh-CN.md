# Matoujie


Matoujie 是一套由 HAE 制作的 Typora 主题，当前仅提供 light 模式，主打清爽、克制、易读的编辑体验。


## 使用方法

1. 在 Typora 中打开 `设置` -> `外观` -> `打开主题文件夹`
2. 将 `matoujie.css` 复制到主题目录
3. 退出 Typora 并重新打开
4. 在 `主题` 菜单中选择 `Matoujie`

## 自定义

- 字体：在 `matoujie.css` 顶部修改 `--font-ui` / `--font-mono`
- 代码字体：默认等宽字体优先使用 `Maple Mono NF CN`
- 排版：`#write` 的宽度、正文行高和标题间距都可以继续微调
